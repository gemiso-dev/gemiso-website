# 고객사 로고 파일

이 폴더에 로고 파일을 넣으면 `/customers` 페이지 그리드에 자동으로 표시됩니다.
파일이 없으면 해당 칸은 회사명 텍스트로 대체됩니다(코드 수정 불필요).

## 규칙

- 파일명: 아래 표의 **파일명**을 그대로 사용 (`<slug>.png`)
- 형식: **PNG**, 배경 투명 권장
- 권장 크기: 높이 약 92px 이상(가로는 비율 유지). 그리드에서 높이 46px로 표시됩니다.
- 로고는 기본적으로 흑백(grayscale)으로 정리되어 보이고, 마우스를 올리면 원래 색으로 표시됩니다.

> SVG로 제공하고 싶으면 알려주세요. `components/customers-data.ts`의
> `customerLogo()` 확장자만 바꾸면 됩니다.
